系统介绍

电子商务推荐系统

基于 Hadoop + Spark 的电子商务推荐系统

技术架构
  • 大数据平台: Hadoop 3.x + YARN
  • 计算引擎: Apache Spark 3.x
  • 数据仓库: Apache Hive
  • 推荐算法: ALS (交替最小二乘法) 协同过滤
  • Web 框架: Flask
  • 前端框架: Bootstrap 5 + ECharts
数据集

使用 Retailrocket 电子商务数据集：

  • events.csv: 用户行为数据（浏览、加购、购买）
  • item_properties.csv: 商品属性信息
  • category_tree.csv: 商品类目层级
